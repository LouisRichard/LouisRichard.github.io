My first embedded system project

This project was the first where I had the chance to program hardware. The objective was to write the program that would run on the micro controller for a numeric dice. The board was designed by the CPNV. It used two 7 segments displays to print the numbers. The dice has the ability to switch between 4, 6, 8, 10, 12 and 20 sided dices.

To develop this program and test it, the CPNV had us use the PIC16F818 micro controller on the EasyPic 7 development board. They had us solder the components on the PCB prior to programming. It was, overall, a fun project where I had the opportunity to learn a lot about code formatting and using C on embedded systems.

Note : This project was made pretty early on my coding "career". The code is written in french and the all variables are globals. I've since learned. Unfortunatly, there is no github repository for this project but if you want to take a look at the code, feel free to send me an email.
