A project written in C# to organise your game library

This project was made in the context of my studies. It was a group project meant to be done in C# .NET. The subject was free to choose but had to obey some specifications such has : using a database, having some sort of login, or using the Agil method to manage the project.

This project was made with the intention of keeping it free and open source without any restrictions enforced by our school. This proved difficult at times but we still managed to do it using only free softwares except for Visual Studio Entreprise. The objective for doing that was to keep updating the software until we found it to be "good enough". As time went on, we both took other projects and this one was left to the side. I'd personally like to return to it some day.
