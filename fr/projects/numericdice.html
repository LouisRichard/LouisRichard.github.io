CPNV - Ste-Croix, VD, Suisse

Mon premier projet de système embarqué

Ce projet était la première fois que j'ai eu la chance de programmer du hardware. L'objectif était d'écrire un programme qui s'exécuterait sur le microcontrôleur pour un dé numérique. Le circuit à été designé par le CPNV. Il utilise deux affichages 7 segments pour afficher les nombres. Les dés ont la capacité de basculer entre les dés à 4, 6, 8, 12 et 20 faces.

Pour développer ce programme et le tester, le CPNV nous a fait utiliser le micro contrôleur PIC16F818 sur la carte de développement EasyPic 7. Ils nous ont fait souder les composants sur le PCB avant la programmation. C'était, dans l'ensemble, un projet amusant où j'ai eu l'opportunité d'apprendre beaucoup sur le formatage de code et l'utilisation du C sur les systèmes embarqués.

Note : Ce projet a été réalisé assez tôt dans ma "carrière" de codage. Le code est écrit en français et toutes les variables sont globales. J'ai appris depuis. Malheureusement, il n'y a pas de repo github pour ce projet mais si vous voulez jeter un œil au code, n'hésitez pas à m'envoyer un email.
