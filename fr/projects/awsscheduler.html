A project designed to manage existing AWS Infra

Aws Scheduler is designed to manage existing AWS Infra (start, stop, alert, email, budget follow up).
This project was made while in an internship at the CPNV. Mr. Glassey being the product owner and the project manager, he is the one who makes decisions and review the code in the end.
The project is writted in OO-PHP, which was a first for me. I still had a lot of fun working on it while I worked there as part of the team.

Now, the project is still being run by Nicolas Glassey and still being developped by himself and the interns at the CPNV. I truly hope the interns have as much fun on this project as I had. It was a real pleasure working with Mr. Glassey.
